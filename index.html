Setup & API Keys
One-time configuration — about 10 minutes
🔒 How your data stays private
🧠
Session-only storage. API keys are held in memory and cleared automatically when you close this tab. Nothing is written to disk or sent to any server.
🌐
Direct API calls only. Your browser talks directly to Square and Squarespace — no middleman, no proxy, no third-party service sees your data.
📄
Read-only tokens. Square and Squarespace API keys are configured with read-only access. They can pull transaction history — they cannot move money or change settings.
📊
Your Google Sheet, your account. Data is written directly to your Google Drive. The Apps Script URL is a long random token — practically unguessable.
💻
Weakest link: your Chromebook lock screen. Use a strong password and lock your screen when away. This is true of QuickBooks too.
⚡ You'll re-enter keys once per session. This is intentional — it means keys never persist anywhere on your device. On your Chromebook, this takes about 15 seconds to paste back in.
1
Get Your Square API Key
Free, takes 2 minutes. Use a read-only Production token.
  1. Go to developer.squareup.com — sign in with your Square account
  2. Click "Create Application" → name it "SoPo Blooms Sync"
  3. Click the app → go to Credentials
  4. Copy the Production Access Token (starts with EAAAl...)
  5. In Permissions, ensure only Orders Read and Payments Read are checked
2
Get Your Squarespace API Key
Enable Commerce read-only access only.
  1. Log into Squarespace → Settings → Advanced → Developer Tools → API Keys
  2. Click "Generate Key", name it "SoPo Sync"
  3. Under permissions, check only Commerce → Orders (Read)
  4. Copy the key and paste below
3
Connect Your Google Sheet
Deploy a free Google Apps Script that receives data and appends it to the correct tab.
  1. Open your SoPo Blooms Financial Tracker in Google Sheets
  2. Click Extensions → Apps Script
  3. Delete existing code, paste the script below, click Save
  4. Click Deploy → New Deployment → Web App
  5. Set "Who has access" → Anyone, click Deploy, authorize
  6. Copy the Web App URL and paste it below
function doPost(e) {
  try {
    var data = JSON.parse(e.postData.contents);
    var ss = SpreadsheetApp.getActiveSpreadsheet();
    var sheet = ss.getSheetByName(data.tab) || ss.getSheetByName('\ud83d\udcca Master Ledger');
    var rows = data.rows;
    for (var i = 0; i < rows.length; i++) {
      sheet.appendRow(rows[i]);
    }
    return ContentService
      .createTextOutput(JSON.stringify({status:'ok', added: rows.length}))
      .setMimeType(ContentService.MimeType.JSON);
  } catch(err) {
    return ContentService
      .createTextOutput(JSON.stringify({status:'error', message: err.toString()}))
      .setMimeType(ContentService.MimeType.JSON);
  }
}
